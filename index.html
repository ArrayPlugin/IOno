iOno is a plugin through which life becomes better Minecraft, for example, if your server is at 1.8 Bukkit probably you can change the mode of survival in the creative, but if your server is only SURVIVAL then this plugin is useful for administrators adds a few useful commands

Copy all unpacked files to plugins folder and restart server.
